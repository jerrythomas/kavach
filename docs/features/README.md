# Features

Per-feature directories containing specs, plans, and task breakdowns.

## Format

Each feature gets its own directory: `NNN-feature-name/`

| File | Purpose |
|------|---------|
| `spec.md` | Feature specification (from `agents/templates/spec-template.md`) |
| `plan.md` | Implementation plan (from `agents/templates/plan-template.md`) |
| `tasks.md` | Phased task breakdown (from `agents/templates/tasks-template.md`) |
