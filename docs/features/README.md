# Features

Module-level features — the **what** and **why**.

## Modules

- [Authentication](01-Authentication.md) — Sign in, sign up, password management, session handling
- [Authorization](02-Authorization.md) — Route protection, role-based access control
- [DataAccess](03-DataAccess.md) — Query filters, CRUD operations, file management
- [UserInterface](04-UserInterface.md) — Auth components, cached logins, smart layout
- [Observability](05-Observability.md) — Structured logging, pluggable writers
- [Verification](07-Verification.md) — Automated checks for multi-platform reliability
- [Publish](08-Publish.md) — Public learn site, documentation, and demo releases
- [Platform Aware Configuration](09-PlatformAwareConfiguration.md) — CLI platform capabilities, smart prompts
